Latitude vs. Weather Analysis (Web Visual Dashboard - WeatherPy)

Project Description

Analysis of various weather-related factors in relation to latitude with data from the OpenWeatherMap API and cities selected randomly from the Citipy library.

Project Details

  • OpenWeatherMap API
  • Python, Matplotlib/Seaborn, Jupyter Notebook
  • Requests, Numpy, Citipy, OpenWeatherMapy libraries
